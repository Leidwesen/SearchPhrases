This is a collection of all Pokemon GO search terms / search phrases / search strings, their uses and translations, their bugs, and related links.

There's no shortages of lists of these phrases on the internet, but all of them are lacking. Even Niantic's official list and the best alternative I could find (Bulbapedia) are missing phrases and functionalities.

This site aims to be the definitive list of all things Pokemon GO search related.

Translations:

For a tool to automatically translate search strings, use Phrase Translator.

To see translations for:

- most phrases, change the language on the official Niantic List.
- all phrases, consult this spreadsheet.

- most moves in most languages, consult this Bulbapedia entry.
- all (in-game) moves in all languages, consult this spreadsheet.

- all pokemon names in most languages, consult this Pokewiki entry.
- most pokemon names in all languages, consult this spreadsheet.

Complexity:

For more complex searches involving parentheses, u/Lebeg134's converter transforms strings into Pokemon GO's syntax.

IVs:

The IV filters introduced in July 2022 can only search in ranges, so tools for finer IV searches can still be useful. They use CP+HP combinations to do so.

PvP IVs offers a powerful tool for creating IV search strings, both PvP and PvE.

Other:

Pogostring is useful for building dex number only strings.

Adding on to pogostring's functionality, Fly33's Tool allows for more complex trade strings (filtering by shiny, gender, and regional form).

I offer a spreadsheet tool for converting dex numbers in strings to Pokemon names.

Trummler12 hosts a collection of various Strings in this document.

Outdated:

u/Trummler12 used to post monthly Community Day strings on reddit.

u/Errubki also used to post monthly community day strings.

Tools such as ark42, rplus, and gohub used to create PvE IV strings, though they are now largely outdated.
