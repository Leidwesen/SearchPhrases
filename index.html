Translations:

For translations of most phrases, change the language on the official Niantic List

For converting from species names to numbers, Pogostring is useful

Complexity:

For more complex searches involving parantheses, they can be converted into Pokemon GO's syntax using u/Lebeg134's converter

IVs:

There is no direct search for IVs, so the best approximations use CP+HP combinations. Tools such as ark42 and gohub help with that. u/Trummler12 maintains spreadsheets of such search strings

PvP IVs:

When searching for good PvP IVs, tools such as PvPIV's provide string generators.
